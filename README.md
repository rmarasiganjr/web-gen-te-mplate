# Vue3 Web App with Authentication

This is a Vue3 web application with user authentication, which includes a landing page, login page, and dashboard. The application uses Bootstrap 5 for styling, Vue Router for navigation, Axios for API calls, Vuex for state management, and cookies for token storage.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

## Features

- Vue3
- Bootstrap 5
- Vuex
- Vue Router
- Axios
- Token stored in cookies
